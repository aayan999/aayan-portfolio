# Aayan - MERN Portfolio

This is a starter Vite + React + Tailwind project containing a single-page portfolio for Aayan.

## Quick start

1. Install dependencies:
   ```
   npm install
   ```

2. Run the dev server:
   ```
   npm run dev
   ```

3. Build for production:
   ```
   npm run build
   ```

The project is ready to be deployed on Vercel or Netlify.
